# API Paths

## Users

```no-highlight
GET/POST/PATCH/DELETE exloc.io/auth/auth-service/users/username{.format}
```

Auth Services:

* [ ] GitHub
* [ ] GitLab
* [ ] BitBucket
* [ ] Twitter
* [ ] Facebook

Supported Formats:

* [ ] html
* [ ] json
* [ ] txt,sh,rb - j/k

* [ ] html

* [ ] git url(?)
